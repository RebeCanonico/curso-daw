Usando flexbox

Para alinear elementos usando flexbox. Estos deben de estar dentro de un elemento padre (al que vamos a llamar contenedor) con la propiedad display: flex;

Los elementor hijos (que esten dentro) de un contenedor flex se convertiran, automaticamente, en elementos flexibles.
