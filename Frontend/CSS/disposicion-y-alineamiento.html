"Layout" en CSS

El layout es la disposicion o la forma en la que los elementos se muestran en pantalla segun su tipo.

Display

Es la propiedad mas importante de CSS para controlar el alineamiento.

la propiedad display especifica la forma en la que se muestra un elemento.

En HTML cada elemento posee un valor por defecto dependiendo del tipo de elemento que sea. La mayoria de los elementos poseen una propiedad block o inline por defecto.

Diferencias entre display: none;y visibility: hidden;

Al ocultar un elemento con display: none; la pagina se mostrara como si el elemento no estubiera ahí.

Al ocultar un elemento con visibility: hidden el elemento no se mostrará pero el espacio que ocupa quedará en blanco.

La propiedad position

Esta propiedad define el metodo de posicionamiento de un elemento.

Hay 5 tipos de posición.

  • static
  • relative
  • fixed
  • absolute
  • sticky
    •

Los elementos se posicionan con las propiedades top, left , right, bottom. Sin embargo estas propiedades no funcionaran sin especificar primero la position. Ademas, dependiendo del tipo de position las propiedades funcionaran de manera distinta.

position: static

Todos los elementos tienen este valor por defecto.

Las propiedades top, bottom, left y right no afectan a los elementos con posición estática.

position: relative

Un elemento con este valor se posicionará de manera relativa a su posición normal. es decir, el centro del eje sera sobre si mismo.

Este div tiene position: relative

Position: fixed

Un elemento con este valor se posicionará relativamente respecto a la pantalla. Lo que quiere decir que tendra siempre la misma posicion incluso si scrolleamos la web.

position: absolute

Un elemento con este valor se posicionara respecto a su contenedor posicionado (no estatico).

Si un elemento absoluto no tiene ancestros posicionados, usará el body y actuará como un position: fixed.

Este elemento div tiene una position: relative
Este elemento div tiene una position: absolute

position: sticky

Un elemento sticky alternará su comportamiento entre relative y fixed dependiendo de la posicion del scroll de la pagina.
