Grid o rejilla

El sistema grid de CSS permite utilizar una plantilla de rejilla, con filas y columnas, para alinear de manera sencilla paginas web sin necesidad de usar float o posicionamiento.

Elementos grid

Una rejilla consiste en un elemento padre con uno o mas hijos.

El elemento padre debe tener la propiedad display: grid o inline-grid. Los elementos descendientes se convertiran automaticamente en objetos(items) grid.
