Media Queries

En CSS3, se introdujo la idea de que en vez de adaptar las webs segun el tipo de dispositivos desde el que se esta visualizando, las adaptaremos segun las capacidades.

Esto quiere decir que en vez de revisar si el dispositivo es una pantalla, un móvil, una impresora, etc... Simplemente nos basaremos en los siguientes parametros:

  • Ancho y alto del viewport
  • Ancho y alto de la pantalla del dispositivo
  • Orientación - si el dispositivo está en vertical u horizontal
  • Resolución

Usar media queries nos capacita con la posibilidad de hacer paginas hechas a medida para pantallas de escritorio, portatiles, tablets, moviles, etc...

Sintaxis

Una expresión de media query puede contener una o varias condiciones que se resuelven en true o false. 

            @media not|only mediatype and (condiciones){
                Código CSS
            }
