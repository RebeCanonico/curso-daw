Selectores de CSS

Los selectores tienen la función de encontrar o selecionar elementos HTML a los que queremos dar estilo.

  • Selectores simples o sencillos - Apuntan a elementos basándose en su clase (.), id (#) o nombre h1.
  • Combinadores - Selectores basados en una especifica relacion entre los elementos .contenedor div.
  • Pseudo-clases - Son selectores basados en el estado del elemento. button:hover
  • Pseudo-elementos - Selecciona una parte especifica de un elemento.
  • De atributo - Seleciona elementos basandose en un atributo o en el valor de un atributo.

Selectores de nombre de elemento

Selecciona los elementos basandose en su nombre de elemento.

Ejemplo: Todos los span serán de color morado. 

                span {
                    color: purple;
                }

Selectores de id

Este selector usa el atributo id de un elemento HTML para seleccionar un elemento especifico.

La id de un elemento es unica en el documento. por lo tanto, el selector de id # se usa para especificar un unico elemento.

Los selectores pueden especificar elementos dentro de una clase.

El selector universal

El selector universal * selecciona a todos los elementos.

Selector de agrupación

Sirve para agrupar aquellos selectores que tienen las mismas caracteriscicas.

Ejemplo: Digamos que queremos darle la misma sombra que pSombras a todos los h1 y h2. En vez de ir a todos los elementos h1 y h2 y darles esta clase podemos hacer lo siguiente: 

        
            .pSombras {
                box.shadow: 5px 5px 3px -2px;
            }
            h1 {
                box.shadow: 5px 5px 3px -2px;
            }
            h2 {
                box.shadow: 5px 5px 3px -2px;
            }

Agrupado quedaria así:

        
            .pSombras, h1, h2 {
                box.shadow: 5px 5px 3px -2px;
            }
