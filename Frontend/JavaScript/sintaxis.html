Sintaxis de JavaScript

La sintaxis es la seria de reglas que define cómo se construyen los programas. Ref al js línea 1.

Valores de JavaScript

Hay dos tipos de vlaores: fijos y variables

A los fijos se les llama literales (literals).

A los variables se les llama Variables.

Valores fijos, literales

LOs dos tipos de literales son los numeros y los strings, o cadena de caracteres.

Los números se escriben con o sin decimales.
10.50
1001

Strings son texto, puede ir escrito entre comilla dobles o simples.
"Jorge González"
'Jorge González'

Variables

EN programacion las variables se usan para almacenar valores.

JavaScript se sirve de las keywords var, let, const para declarar variables.

El signo = se usa para asignar valores a las variables.

Operadores

JavaScript usa operadores aritmeticos (+ - * /) para calcular valores.

Expresiones

Las expresiones son una combinacion de valores, variables y operadores que computan en un resultado o valor. ref al js línea 11

COmentarios

Los comentarios son líneas de código que son ignoradas. No se ejecutan.

Tienen principalmente dos usos. EL primero es explicar el código y el segundo es para desactivar cierta parte de nustro código sin necesidad de borrarla.

EN JavaScript hay dos maneras de comentar. EN línea con // y en bloque con /*...*/

Identificadores/nombres

LOs identificadores se usan para nombrar variables y funciones.

HAy una serie de reglas para nombrar comunes en la mayoria de lenguajes de programación.

Un identificador debe emperar por:

  • Una letra (A-Z o a-z)
  • Un simbolo de dólar ($)
  • o una barra baja _

Los caracteres siguientes pueden ser numericos, letras, _, $.

No se permite emnpezar con números!

Uso de mayúsculas

Los identificadores son sensibles a las mayusculas. esto quiere decir que lastName y lastname son dos variables diferentes.

Y, por ejemplo, JavaScript no interpreta LET o let como la palabra clave let.

Camel Case

Antes, los programadores usaban guiones - para unir distintas palabras en un identificador. last-name; first-name

En JavaScript los guiones estan reservados para las restas. Por lo que los programadores de JS se inclinan a separar las distintas palabras mediante mayusculas, dejando el resto en minusculas.

primerApellido, colorVehiculo, codigoPostal.

COmo se ha podido apreciar, no es conveniente el uso de tildes u otros caracteres ajenos al ingles, lenguaje en el que se ha diseñado la programación.
