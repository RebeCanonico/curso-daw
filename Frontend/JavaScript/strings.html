String con JS

Un string es cero o más caracteres escritos dentro de comillas.

Esto lo convierte básicamente en una cadena de caracteres.

Se puede usar tanto comillas simples como dobles. Un string puede contener comillas siempre y cuando sean distintas de las que rodean el string:

let str = 'Tere "Terere" Herrera'

Si imprimimos este string: Tere "Terere" Herrera.

Métodos de string

.length

Para obtener la longitud del string se usa el metodo length().

let txt = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'
let longitud = txt.length

slice()

Corta o estrae una parte de un string y la retorna como nuevo string.

slice() necesita dos parametros. La posición inicial y la final (no incluidas en el corte).

Ref al js línea 9

let corte = txt.slice(15, 19);
let pCorte = document.getElementById('pCorte');
pCorte.innerHTML = corte;

substr()

Funciona similar a slice(), pero el segundo parametro indica la longitud del String a recortar. Deprecado

replace()

Reemplaza una parte de un string y lo devuelve como un string nuevo.

EL primer parametro es el valor a buscar en el string y el segundo es el valor por el que lo reemplazamos.

Estos valores son case sensitive. Es decir, que los valores de mayúsculas y minúsculas deben coincidir.

Sólo se reemplaza la primera coincidencia en caso de haber varias en el string.

Para sortear la diferencia de las mayúsculas, se puede usar una expresión regular.

Transformar a minúsculas y viceversa

Podemos usar toLoverCase() o toUpperCase() para transformar un string o una parte en minúsculas o mayúsculas, respectivamente.

Limpiar strings

Podemos usar trim() para limpiar los espacios en blanco redundantes de un string.

Si tenemos un str nombre = " Román" el trim lo dejará como "Román".

charAt()

Retorna el cáracter ubicado en el índice especificado como parámetro.

Convertir string en arrays con split()

Si tenemos un string cuyos valores estan separados pot comas o cualquier otro caracter distinguible, podemos servirnos de esto para transformarlo en un array.
