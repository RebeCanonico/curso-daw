Listas HTML

Listas sin orden

Se usa la etiqueta <ul> como contenedor. Cada elemento de la lista irá en una etiqueta <li>

Se usan para mostrar listas en la que el orden de los elementos no es importante. Por ejemplo: La lista de la compra, de asistencia, de material escolar.

  • fruta
  • pasta
  • carne
  • Huevos

Listas ordenadas

Se usa la etiqueta <ol> (orderd list) como contenedor. cada elemento de la lista ira en una etiqueta <li>

Son listas numeradas. Se usan cuando el orden es importante y los elementos tienen importancia segun el orden o son pasos a seguir, por ejemplo.

Algunos ejemplos de listas ordenadas son: una receta

Receta para cafe con leche

  1. Preparar el café
  2. Añadir la leche caliente
  3. Añadir azucar al gusto
  4. servir

LIstas descriptivas

Son listas de terminos, cada uno de estos con su propia descripción. Se usa la etiqueta <dl> (description list) como contenedor, cada termino irá en un <dt> y cada descripción en un <dd>

Café
- Bebida de color negro que se bebe caliente.
Leche
- Bebisda de color blanco que se bebe fría o caliente.
