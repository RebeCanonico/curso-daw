Id's y Clases

Las id y las clases son atributos que se pueden asignar a los elementos de HTML. Mientras que las clases se pueden asignar a varios elementos, las id son únicas y sólo puede tenerlas un elemento.

Clases

La principal función de las clases es apuntar a varios elementos mediante CSS o la manipulación mediante JavaScript a múltiples elementos.

Ejemplo:

Este texto es importante

Este texto es importante

Los elementos también pueden tener múltiples clases. Sólo deben estar separadas por un espacio cuando se les asigna el atributo.

Este texto es importante y está en cursiva.

id's

Los Id's apuntan a un único elemento. No puede haber mas de un elemento en la misma id.

Sintaxis: En CSS se declara con una almoadilla #. No puede empezar por un numero y no puede contener espacios en blanco.
