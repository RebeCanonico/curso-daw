Prueba de JavaScript

Si el HTML proporciona la estructura del documento, el CSS el estilo visual, entonces es JS el que proporciona la interactividad de la web con el usuario.

Con JS podemos hacer que la pagina reacione ante las distintas acciones del usuario. Hacer clic, pasar el raton sobre un objeto en concreto, hacer scroll, usar el teclado, etc.

Al igual que HTML nos provee de la etiqueta stylepara onsertar el CSS. Tenemos la etiqueta script para escribir JavaScript. El código también puede incorporarse como atributo de los elementos (como sucede con el atributo style) por lo que su uso dependerá del criterio de desarrollador.
